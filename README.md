# Nrtur Mobile Design

A high-fidelity mobile web application prototype and design showcase for the **Nrtur** platform.

This repository contains the interactive mobile prototype, packaged as a fully self-contained HTML document (index.html) featuring CSS styles, layouts, assets, and React component code.

---

## 🎞️ Features

- **Modern Mobile UI/UX:** A sleek and interactive interface designed specifically for mobile viewports.
- **Self-Contained Build:** Includes all necessary assets, scripts, stylesheets, and data files bundled directly in index.html for offline viewing and easy distribution.
- **Clean Responsive Layout:** Beautiful typography, harmonized color palettes, and interactive transitions.

---

## ⚡ Getting Started

To view and interact with the mobile design prototype:

1. **Clone the Repository:**
   `ash
   git clone https://github.com/Nrtur/nrtur-mobile-design.git
   cd nrtur-mobile-design
   `

2. **Open the Design:**
   Simply open the index.html file in any modern web browser:
   - **Double-click** the index.html file in your file explorer.
   - Or open it via terminal:
     - **macOS:** open index.html
     - **Windows (CMD):** start index.html
     - **Linux:** xdg-open index.html
